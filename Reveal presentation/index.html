Key Features of React

2) Component-Based Architecture
3) JSX (JavaScript XML)
4) Unidirectional Data Flow

React's Component Lifecycle

Mounting: This phase occurs when a component is being created and inserted into the DOM.

Updating: When a component's state or props change, it goes through the updating phase, triggering a re-render.

Unmounting: In this phase, a component is removed from the DOM.
